# 邯郸市农业气象数据分析
基于Python对邯郸市全年气象数据进行全流程分析，包含数据清洗、时序分析、相关性分析、季节特征分析与可视化展示。

## 项目内容
- 气象数据清洗（缺失值、异常值、重复值处理）
- 时间序列趋势分析（温度、降水、湿度、气压、风速）
- 多指标相关性分析与热力图可视化
- 季节分布、月度变化、极值分析
- 农业气象关联分析与结论输出

## 技术栈
Python、Pandas、NumPy、Matplotlib、Seaborn

## 项目成果
- 完成365天全量气象数据标准化处理
- 输出20+专业可视化图表（折线图、箱线图、热力图、雷达图、日历图等）
- 形成完整农业气象分析报告

  
## 实验概述
本实验基于Python编程语言，对邯郸市2024年农业气象数据进行爬取、预处理、分析和可视化，旨在揭示气象要素的时空变化规律及其对农业生产的影响。

## 数据准备

### 原始数据
1. 原始数据包含12个月的Excel文件，命名格式为：`weather_data (0).xlsx` 到 `weather_data (11).xlsx`
2. 数据来源：中国气象数据网
3. 数据字段包括：
   - 日期、天气现象、风力风向
   - 最低温度、最高温度、平均温度（单位：°C）
   - 湿度（%）、降水量（mm）、日照时长（h）、短波辐射总量

## 实验步骤

### 步骤1：数据预处理
1. 运行数据预处理脚本：
   python 数据预处理.py
2. 脚本执行过程：
   - 读取12个月的Excel文件
   - 数据清洗和格式标准化
   - 合并数据并处理缺失值
   - 生成完整的年度数据集：`邯郸市24年农业气象数据.csv`

3. 输出：
   - 生成CSV文件包含完整的2024年气象数据

### 步骤2：数据可视化分析
1. 运行可视化分析脚本：
   python 绘图.py
2. 脚本执行过程：
   - 读取预处理后的CSV数据
   - 提取时间特征和风力等级
   - 生成多种可视化图表

3. 生成的可视化结果并保存：

   **折线图、散点图等：**
   - `温度趋势图.png` - 全年温度变化趋势
   - `月度温度分布箱线图.png` - 各月温度分布
   - `温度与降水量关系散点图.png` - 气象要素相关性
   - `四季气象指标雷达图.png` - 季节特征对比
   - `月度降水量柱状图.png` - 降水分布
   - `2024年温度波形图.png` - 温度曲线
   - `月度日照时长变化图.png` - 日照时长月度变化
   - `日照时长与温度关系图.png` - 日照与温度关系
   - `日照时长与降水量关系图.png` - 日照与降水关系
   - `四季日照时长分析图.png` - 季节日照对比

   **日历图：**
   - `气温.html` - 温度日历图（创新点）
   - `降水.html` - 降水日历图（创新点）

   4.基于生成的图表对邯郸市气象情况进行分析
